 - `expand`/`unexpand`(tabs=>spaces, spaces=>tabs)

 - `PATH`: Path that searched for executables
 - `PS1`,`PS2`,`PS3`,`PS4`
 - `PWD`: current working directory
 - `HOME`: home directory of the current user
